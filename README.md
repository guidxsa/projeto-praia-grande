# Projeto Extensionista EMCEF Praia Grande

## Resumo

Esse é um projeto extensionista da matéria de PI 5 da faculdade UCL que tem como objetivo ajudar a escola EMCEF Praia Grande, sem fins lucrativos e totalmente gratuito.

A escola apresentou queixas de que o modo de informar os pais era muito ultrapassado e gastava muito papel, o que era um grande problema levando em conta que gerava custos para escola e não era eficiente.
Foi proposto um projeto de criação de um site público informativo gerenciado pelas autoridades da escola.

O sistema consiste em ser um elo entre as autoridades da escola e os alunos/responsáveis, facilitando a comunicação e a tornando mais eficiente.

## Cronograma

### Semana 0:
- Apresentacao inicial do projeto
- Escolha da instituicao alvo
- Entendimento da necessidade da escola
- Coleta de requisitos
- Configuração do Repositório no GitHub e convite para o grupo
  
### Sprint 1:
- Criação do Protótipo no Figma
- Termo de intenção
- Escolha da tecnologia
  
### Sprint 2:
- Definição do modelo de dados
- Configurar a hospedagem gratuita
- Desenvolver a página principal

### Sprint 3:
- Configurar o Banco de Dados
- Implementar o Sistema de Autenticação
- Desenvolver o formulário administrativo para criar avisos

### Sprint 4:
- Implementar o upload de dados
- Implementar abas de diferentes informes

### Sprint 5:
- Implementar filtros por data e por tipo de aviso
- Implementar exclusão e edição de avisos

### Sprint 6:
- Testes de usabilidade
